# Important Skills for a Software Engineer

## Full-Stack (Frontend and Backend)

1. [Git](Useful-Git-Commands.md)
   1. Understanding at least at a basic level commands for Version Control is critical
2. Linux commands
   1. Extremely important whenever working on MacOS or Linux
3. Docker
   1. Useful for containerizing applications
4. Vibe coding with AI LLM
   1. Boosts productivity exponentially
      1. Used to be reading manuals -> web searching and reading Stack Overflow posts
5. Postgres SQL DB
   1. Having an understand of SQL for holding and querying data is important
6. HTML, CSS & JS
   1. Needed for creating and decorating websites
7. JavaScript & TypeScript
   1. Needed for adding logic to websites
8. React or Vue or Angular web-app
   1. Modern high-level JS/TS libraries for generating websites
9. Python
   1. Simple to learn scripting language with lots of power
10. Python Flask API
    1. Using python as a backend API server
11. JS/TS Node API
    1. Using JS/TS as a backend API server
