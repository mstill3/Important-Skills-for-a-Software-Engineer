# Important Skills for a Software Engineer

## Common for all Software Engineers

### Important Skills

| Skill                                                                                          | Reasoning                                                                            |
| ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| [Git](https://www.w3schools.com/git/default.asp)                                               | Understanding at least at a basic level commands for Version Control is critical     |
| [SCRUM / Agile Process](https://www.parabol.co/resources/scrum-101-guide)                      | How the modern software industry develops code                                       |
| [Markdown](https://www.markdownguide.org/basic-syntax)                                         | Understand note taking for software developers                                       |
| [Linux commands](https://linuxize.com/cheatsheet/linux-commands/linux-commands-cheatsheet.pdf) | Extremely important whenever working on MacOS or Linux                               |
| [Vibe coding with AI LLM](https://vibecodng.org)                                               | Boosts productivity exponentially; Modern version of reading the Coding manual       |
